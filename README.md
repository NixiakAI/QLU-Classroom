只需要如下两步：
1. 在```get_schedule.py```文件中修改开学时间（开学那周的周一）
```python

year,month,day=2024,2,20
```
2. 在 ```get_course_on_table.py```中提供在 [教务系统](http://jwxt-qlu-edu-cn.vpn.qlu.edu.cn/) 中的cookie，并且运行一次获得数据包
```python
# <<<<<!!!需要使用代理，提供Cookie获取数据!!!>>>>>
Cookie = "抓取提供"
```
